ESC190: Data Structures and Algorithms (Winter 2023)

Course staff

Course description   This course serves as an the second half of the introduction to computer programming and computer science. We will introduce advanced features of the Python programming language, and introduce the C programming language. We will implemente data structures and algorithms in Python and C, and analyze them. We will discuss good practices in software engineering (designing and building large software systems). We will analyze the efficiency of algorithms, and discuss designing efficient algorithms. Various research areas in computer science (AI, software engineering research, the theory of computation, etc.) will be introduced throughout the term.

Labs

  • Lab 1: VS Code, Introduction to C, the C memory mode. Solutions
  • Lab 2: Pointers, strings, and Binary Search Deluxe. Solutions (except for Binary Search Deluxe, which is part of Project 1)
  • Lab 3: structs, blocks of structs, gcc, and header files. Solutions
  • No lab assignment during the week of Feb 6, but TAs will be available during lab time to help with Project 1. Attendance is optional.
  • Lab 4: Valgrind and the C and Python memory models. Solutions: lab4.c, lab4.py
  • Lab 5: Linked lists and ArrayLists. Download LL.c. Solutions: main.c, list.h, linkedlist.c, arraylist.c
  • Lab 6: Implementing Queues. Download: Queue.py. Solutions: CircularQueue.py, cirqueue.c
  • Lab 7: Dynamic programming, image files, and implementing Heaps. Using image files in Lab 7. Download: heaps.py. Solutions: brighten.c, heap.c
  • Lab 8: Continue working on the Labs 7-8 handout. You must show your work to a TA both on the week of Lab 7 and on the week of Lab 8.
  • Lab 9: Dijkstra's Algorithm in C. dijkstra.py, lab09.c. Solution: lab09_soln.c
